# Windows-Scripts

[![Download Latest Release](https://img.shields.io/github/v/release/UniFy-Endpoint/Windows-Scripts?label=Download%20Latest&style=for-the-badge&logo=github)](https://github.com/UniFy-Endpoint/Windows-Scripts/releases/latest)

General Windows PowerShell utilities: run commands as SYSTEM via PsExec, and clean Win32 app registry entries.

## Files

- PsExec_As_System.ps1
- Win32Apps-RegistryWipe.ps1

## Requirements

- PowerShell 5.1 or later
- Appropriate Microsoft 365 / Intune admin permissions
